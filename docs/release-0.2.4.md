# NovelX Desktop 0.2.4

## 主要修复

- 大管家可以读取、搜索和总结当前项目中的真实文件，不再把六个领域分类误报为文件或项目。
- 支持 Markdown、文本、代码、DOCX 和 EPUB；图片及其他二进制返回可审计元数据。
- 普通项目文件可通过 Change Set 创建、覆盖和删除。
- 覆盖使用 SHA-256 并发保护，避免 Agent 覆盖用户刚刚修改的内容。
- Agent 修改的真实文件进入检查点版本链，可以随项目版本恢复。
- “已处理”展开项显示实际读取的相对文件路径和完整/片段状态。
- 修复 DeepSeek 不接受顶层联合工具 Schema 导致的 HTTP 400。
- 修复专项 Agent 结构化结果工具偶发不调用和无效提交计数错误。

## 验证

- TypeScript 类型检查通过。
- 76 个测试文件、292 项测试通过。
- DeepSeek 真实 Prompt 评测 10/10 通过。
- 真实 Electron E2E 验证读取、审计、Assist 写入和检查点恢复。
- Windows NSIS 安装包与自动更新元数据生成成功。

## 已知限制

- 安装包暂未配置受信任的 Windows 代码签名。
- 文件审计项暂不能点击跳转到具体编辑行。
